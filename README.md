# OleSpace

OleSpace is a social media web application built using vanilla JavaScript and the Noroff Social API.  
Users can register, log in, create posts, interact with other users, and manage their own content.

---

## Features

### Core Features

- Register new user
- Login as existing user
- View all posts (feed)
- View a single post
- Create a post
- Edit your own posts
- Delete your own posts
- View another user's profile and posts
- Follow / Unfollow users
- Search posts
- View your own profile

### Additional Features

- Comment on posts
- Delete your own comments
- Mobile-friendly navigation (hamburger menu)
- Custom utility CSS framework (OleWind)

---

## Tech Stack

- JavaScript (ES6 modules)
- Vanilla JavaScript (no frameworks)
- Custom CSS framework (OleWind)
- Noroff Social API
- Deployment via Vercel, Netlify or GitHub Pages

Navigate into the project:

- Open index.html in your browser  
  or
- Use Live Server in VSCode

## Authentication

This project uses Noroff API authentication:

- Register via /auth/register
- Login via /auth/login
- API key is created for authenticated requests

---

## API

This project uses the Noroff Social API.

Endpoints used include:

- Posts
- Profiles
- Comments

Documentation:
https://docs.noroff.dev/docs/v2/social/
